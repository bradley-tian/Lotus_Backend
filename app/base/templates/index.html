Lotus Finance API


How Does It Work?

The Lotus Finance API enables investors and learners to better comprehend and apply technical analysis.

With Lotus's retrospective analysis models, users can design various technical strategies, apply them in paper-money simulations, and thus test their efficacy with various stocks and timeframes.

Understanding the past helps us better tread in the future.

Lotus Finance is here to help you identify patterns, build ideas, and perfect strategies.


API Documentation


Retrieving Stock Quote

Retrieving closing price records of a Stock.

HTTP Request URL: [GET] www.lotusfinanceapi.org/api/quote

Parameters:

Name Type Description Examples
ticker String The ticker of the stock to retrieve closing price records from. TSLA, AMD, RBLX, BAC, SPY
start_date String The start date of the historical period. 01-01-2022, 12-30-2005
end_date String The end date of the historical period. 01-01-2022, 12-30-2005
period String The period leading up to present day. Can replace start & end dates. 1d,5d,1mo,3mo,6mo,1y,2y,5y,10y,ytd,max
interval String The price intervals to be retrieved throughout the specified period. 1m,5m,15m,30m,1h,1d,5d,1wk,1mo,3mo

Response Object:

Dictionary containing (timestamp, value) collection of closing prices of the specified stock.


MACD Indicator Analysis Model

Backtesting model oriented on MACD crossover detection.

HTTP Request URL: [GET] www.lotusfinanceapi.org/api/macdModel

Parameters:

Name Type Description Examples
ticker String The ticker of the stock to retrieve closing price records from. TSLA, AMD, RBLX, BAC, SPY
start_date String The start date of the historical period. 01-01-2022, 12-30-2005
end_date String The end date of the historical period. 01-01-2022, 12-30-2005
period String The period leading up to present day. Can replace start & end dates. 1d,5d,1mo,3mo,6mo,1y,2y,5y,10y,ytd,max
interval String The price intervals to be retrieved throughout the specified period. 1m,5m,15m,30m,1h,1d,5d,1wk,1mo,3mo
fast String The fast EMA for the MACD Indicator. 12
slow String The slow EMA for the MACD Indicator. 26
signal String The period to signal for the MACD Indicator. 9
shares String The number of shares in the stock to begin the backtest simulation with. 64, 500, 1000
starting_price String The average price that the starting shares are bought at. 165.87, 200, 3316.2
cash String The amount of auxiliary cash to begin the backtest simulation with. 165.87, 200, 3316.2

Response Object:

Dictionary containing (timestamp, value) pairs of MACD lines, MACD histogram, MACD signal line, Buy/Sell Triggers, and Net Returns.


RSI Indicator Analysis Model

Backtesting model oriented on momentum-based market reversal detection.

HTTP Request URL: [GET] www.lotusfinanceapi.org/api/rsiModel

Parameters:

Name Type Description Examples
ticker String The ticker of the stock to retrieve closing price records from. TSLA, AMD, RBLX, BAC, SPY
start_date String The start date of the historical period. 01-01-2022, 12-30-2005
end_date String The end date of the historical period. 01-01-2022, 12-30-2005
period String The period leading up to present day. Can replace start & end dates. 1d,5d,1mo,3mo,6mo,1y,2y,5y,10y,ytd,max
interval String The price intervals to be retrieved throughout the specified period. 1m,5m,15m,30m,1h,1d,5d,1wk,1mo,3mo
window String The window period measured by the RSI indicator. 14
lower String The low-level line of the RSI indicator. 30
higher String The high-level line of the RSI indicator. 70
shares String The number of shares in the stock to begin the backtest simulation with. 64, 500, 1000
starting_price String The average price that the starting shares are bought at. 165.87, 200, 3316.2
cash String The amount of auxiliary cash to begin the backtest simulation with. 165.87, 200, 3316.2

Response Object:

Dictionary containing (timestamp, value) pairs of RSI values, Buy/Sell Triggers, and Net Returns.
